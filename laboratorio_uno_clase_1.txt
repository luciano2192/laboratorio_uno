getch() y getche() se encuentran definidas en el fichero de cabecera: 
"conio.h" (librería de Borland) y la función getchar() se encuentra 
definida en "stdio.h"

La función getch(): lee caracter a caracter pero no muestra por pantalla 
el carácter leido.

EJ: variable = getch();

La función getche(): lee carácter a carácter y muestra por pantalla 
el carácter leido.

La función getchar(): lee caracter a caracter, hace un barrido, tras 
haber detectado fin ( intro ).

	?...ahora vamos a hacerla con las 3 etapas

	PROTOTIPO: 
	Siempre antes del main, para luego poder invocarla.
	No hace falta declarar el nombre de los parametros,
	puedo declarar solo el tipo de dato.
	
		int suma( int operador1 , int operador2 );


	DEFINICION:
	siempre abajo del main
	
		int suma( int operador1 , int operador2 ) {
			
			int resultado;
			resultado = operador1 + operador2;
			return resultado;
		}
